# rxfirebase
ReactiveX extensions for Firebase #ReactiveX #Firebase #Android

# How to use
>Add this to your project's project level build.gradle file
```
allprojects {
  repositories {
    ...
    maven { url 'https://jitpack.io' }
  }
}
```
>Add the dependency
```
dependencies {
  implementation 'com.github.sooakim:rxfirebase:X.X.X'
}
```
